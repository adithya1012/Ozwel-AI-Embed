üì¶ Script Tag Integration

Best for: Native integration, custom styling, same-domain deployment

Simple Integration:

<!-- Include the chatbot script --> <script src="https://your-domain.com/ozwel-ai-chatbot.js"></script> <!-- Initialize the chatbot --> <script> document.addEventListener('DOMContentLoaded', function() { if (window.OzwelAI) { window.OzwelAI.init(); } }); </script>

Advanced Configuration:

<script> document.addEventListener('DOMContentLoaded', function() { if (window.OzwelAI) { // Custom initialization const chatbot = window.OzwelAI.chatbot; // Configure API key programmatically localStorage.setItem('openai_api_key', 'your-api-key'); // Initialize window.OzwelAI.init(); } }); </script>
